## 概要

「Octopus」は、エンジニアが効率的に学習を進め、必要なスキルを体系的に習得するための スキルナビゲーションツール です。 

資格取得や技術学習における ゴール設定・ロードマップ作成・タスク管理・進捗可視化 を一元化し、  

「何から学べばいいか分からない」「計画を立てるのが苦手」という課題を解消することを目的としています。  

複数の技術領域へ触手を伸ばす“Octopus（タコ）”のモチーフを採用し、 

エンジニアが複数スキルを横断的に伸ばしていく学習イメージを表現しています。  


### ◉　開発背景
>
>エンジニアの学習は範囲が広く、何から取り組むべきか分かりづらい課題があります。  
>
>資格学習では特に、必要なスキルやタスクを「正しい順序」で学び進める必要があり、計画立案が苦手な学習者ほど挫折しやすくなっています。  
>
>Octopus はその問題を解決するために、
>
>**[ゴール → 必要スキル → タスク分解 → ロードマップ化 → 日々の学習ログ]**  
>
>という流れを一つのシステムで完結できるように設計されています。

### ◉　Octopusが解決する課題

>- 何から学べばいいか分からない問題
>- 必要なスキルの全体像が見えない問題
>- ゴールに向けた計画をたてられない問題  

### ◉　ターゲットユーザー

>- エンジニアを目指す学習者
>- IT資格の取得を目指す人
>- 新人エンジニアや初学者
