## 概要

__<ins>「学習は記録すれば、成長に変わる」</ins>__  

 SkillTrackerは、日々の学習をデータとして蓄積し、自分の成長を客観的に可視化できるスキル成長支援プラットフォームです。  
 どれだけ努力を積み重ねても、  
 - 何をどれだけ学んだのか
 - どこが伸びて、どこが弱点なのか
 - 自分は本当に成長しているのか

は記録しなければ見えません。  
SkillTrackerは、学習内容・学習時間・技術カテゴリを一元管理し、折れ線グラフ・棒グラフ・レーダーチャートを用いて成長を見える化することで、  
学習者が自身の成長を実感しながら継続できるよう設計をしています。  

### ○<ins>SkillTrackerが解決する課題</ins>

- 学習しても成長が見えず、継続しにくい
- どの技術に時間を投資したか分からない
- モチベーションが長続きしない  

### ○<ins>ターゲットユーザー</ins>

- 自己学習の習慣を定着させたい人
- 自己成長を可視化したい人
- 資格学習者
