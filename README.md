## 概要

「Octopus」は、エンジニアが効率的に学習を進め、必要なスキルを体系的に習得するための スキルナビゲーションツール です。 

資格取得や技術学習における ゴール設定・ロードマップ作成・タスク管理・進捗可視化 を一元化し、  

「何から学べばいいか分からない」「計画を立てるのが苦手」という課題を解消することを目的としています。  

複数の技術領域へ触手を伸ばす“Octopus（タコ）”のモチーフを採用し、 

エンジニアが複数スキルを横断的に伸ばしていく学習イメージを表現しています。  


### ⚫︎　開発背景
>
>エンジニアの学習は範囲が広く、何から取り組むべきか分かりづらい課題があります。  
>
>資格学習では特に、必要なスキルやタスクを「正しい順序」で学び進める必要があり、計画立案が苦手な学習者ほど挫折しやすくなっています。  
>
>Octopus はその問題を解決するために、
>
>**[ゴール → 必要スキル → タスク分解 → ロードマップ化 → 日々の学習ログ]**  
>
>という流れを一つのシステムで完結できるように設計されています。

### ⚫︎　Octopusが解決する課題

>- 何から学べばいいか分からない問題
>- 必要なスキルの全体像が見えない問題
>- ゴールに向けた計画をたてられない問題  

### ⚫︎　ターゲットユーザー

>- エンジニアを目指す学習者
>- IT資格の取得を目指す人
>- 新人エンジニアや初学者

  
## 機能  

**⚫︎ 学習ロードマップ作成機能**  
```
資格や習得したい技術を選択し、「目標日」「1日の学習可能時間」「休日」などの個人設定に応じた最適な学習計画を自動生成します。
　・必要学習量の自動計算
　・毎日のタスク分解
　・バッファ日の自動挿入
　・期限重視／学習量重視などのモード選択
```
**⚫︎ 学習計画の可視化機能**  
```
作成した計画を視覚的に確認することができます。
　・日ごとのタスク 　　　　　→ 週表示カレンダー
　・全体の進捗やスケジュール → ガントチャート
直感的に「今何をやるべきか」がわかります。
```
**⚫︎ データ管理機能**
```
以下のデータを登録／参照／更新／削除できます。
　・学習計画
　・1日の細かなタスク
　・資格テンプレート（学習項目・必要時間のプリセット）
ユーザーは複数の学習計画を保存・切り替えできます。
```
**⚫︎ ログイン機能**
```
ユーザーごとにデータを安全に保持するための認証機能を提供します。
　・ログイン／ログアウト
　・パスワード管理
　・ユーザー固有の計画データ保存
```
**⚫︎ ユーザー情報登録機能**
```
計画作成に必要なユーザー固有の情報を登録できます。
　・1日の学習可能時間
　・休日設定
　・お気に入り資格テンプレート
これらはロードマップ生成時に反映されます。
```
